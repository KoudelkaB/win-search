# File Search Manager 도움말

`F1`은 앱 안에서 이 도움말을 열고 `F12`는 NTFS 색인을 새로 고칩니다.

## 필터

- `report`는 이름에 포함, `:report`는 시작, `report:`는 끝, `:report:`는 정확히 일치합니다.
- `pdf|docx`는 대안 중 하나이며 공백으로 구분한 항목은 **AND**입니다.
- `src\`는 부모 폴더, `src\\`는 전체 경로를 검색합니다.
- `"C:\Work"`는 바로 아래를, `"C:\Work\\"`는 재귀적으로 검색합니다.

공백이 있는 경로는 따옴표로 묶습니다. `Ctrl+Left`/`Ctrl+Right`는 기록, `Down`은 제안을 표시합니다.

## 검색 및 키

텍스트를 입력하고 `Enter`를 누릅니다. `UTF-8`/`UTF-16`은 텍스트, `HEX`는 `48 65 6C 6C 6F` 같은 바이트를 찾습니다. `Delete` 삭제, `F2` 이름 바꾸기, `O` 열기, `U`/`Z` 압축 풀기/만들기입니다. `›`는 하위 메뉴, `Backspace`는 뒤로, `Esc`는 취소입니다.
