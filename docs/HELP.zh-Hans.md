# File Search Manager 帮助

`F1` 在应用内打开本帮助；`F12` 刷新 NTFS 索引。

## 筛选语法

- `report` 表示名称包含；`:report` 表示开头；`report:` 表示结尾；`:report:` 表示完全匹配。
- `.pdf:|.docx:` 匹配以这些扩展名结尾的名称；以空格分隔的词按 **AND** 组合。
- `src\` 搜索父文件夹；`src\\` 搜索完整路径。
- 名称锚点会应用于路径的每个组成部分，因此 `:src:\\` 可匹配路径中任意位置包含名称恰为 `src` 的文件夹的项目。
- `"C:\Work"` 搜索直接内容；`"C:\Work\\"` 递归搜索子项。

包含空格的路径请加引号。`Ctrl+Left`/`Ctrl+Right` 浏览历史，`Down` 显示建议。

## 实用示例

- **释放磁盘空间** — 清空**筛选**以显示所有已建立索引的项目。点击**大小**列标题（必要时再次点击），直到最大的项目位于顶部，然后从上到下检查列表；使用 `Shift+Delete` 删除不需要的项目（永久删除，不经过回收站）。
- **实时查看应用写入位置** — 清空**筛选**并点击**修改时间**列标题，直到最近修改的项目位于顶部。随后启动或使用要观察的应用；它正在写入的文件会在每次变化后移到顶部，**文件夹**列会显示位置。如果只想观察文件系统的一部分，请先在**筛选**中输入路径。
- **搜索源文件中的文本** — 例如使用 `C:\Projects\\ .cs:` 将结果限制为项目中的 `.cs` 文件。在**搜索**框中输入要查找的文本，然后按 `Enter`。

## 搜索和快捷键

输入文本后按 `Enter`。`UTF-8` 和 `UTF-16` 搜索文本；`HEX` 搜索如 `48 65 6C 6C 6F` 的字节。`Delete` 删除，`F2` 重命名，`O` 打开，`U`/`Z` 解压/创建压缩包。`›` 打开子菜单，`Backspace` 返回，`Esc` 取消。
